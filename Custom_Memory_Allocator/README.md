# Custom Memory Allocator (C++)

A basic custom memory allocator demonstrating low-level memory management
and pointer arithmetic in C++.

## Features
- Manual heap memory allocation
- Controlled memory lifecycle
- Reduced allocation overhead

## Concepts Used
- Pointers
- Heap vs Stack memory
- Memory alignment
- RAII basics

## Build & Run
```bash
g++ src/memory_allocator.cpp -o allocator
./allocator
